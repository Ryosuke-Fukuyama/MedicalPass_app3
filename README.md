# README

## 開発言語
- Ruby version '2.6.5'
- rails version '5.2.5'

## 実行手順
```
$ git clone https://github.com/Ryosuke-Fukuyama/MedicalPass_app3.git
$ cd MedicalPass_app3
$ bundle install
$ rails db:create db:migrate
$ rails db:seed
$ rails s
```
